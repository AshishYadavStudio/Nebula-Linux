Successfully integrated the Calamares graphical installer with KDE Plasma 6 Wayland. Fixed the fatal Qt6/Wayland root crashing bugs and rebuilt the YAML installer branding.

✅ The Birth of Nebula Linux

June 23, 2026

The core Debian Trixie (Debian 13) Live OS was successfully bootstrapped. We set up the initial custom KDE desktop and built the very first bootable ISO image.
